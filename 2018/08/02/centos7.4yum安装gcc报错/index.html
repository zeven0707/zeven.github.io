2018-08-02
本文总阅读量次

[yum install gcc执行报错如下：]

产生该问题的主要原因是：在系统upgrade的时候，残存了上一个版本的软件包

[解决办法：]

首先安装 yum-utils 套件

执行clean duplicate package　　

重新安装glibc

安装完成之后再安装gcc

[linux] linux增加swap空间

1、原有的swap空间不足，安装oracle client报错，需要增加swap

查看当前swap

生成swap文件
dd if=/dev/zero of=/var/swap bs=1M count=1024

将文件格式化为交换文件：
mkswap /var/swap

加载这个交换文件：
chmod 600 /var/swap
swapon /var/swap

添加到开机启动

[linux] shadownsocks-server部署

2019-09-22

1、确保安装 2.6 或者 2.7

2、安装pip

curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
python get-pip.py

3、安装shadowsocks

pip install shadowsocks

查看已安装软件列表
pip freeze|grep shadow
which ssserver
/usr/local/bin/ssserver

创建配置目录配置文件

4、启动服务

ssserver -c /etc/shadowsocks/config.json -d start
启动服务指定日志文件
ssserver -c /etc/shadowsocks/config.json -d start --log-file /tmp/shadowsock.log
在后台启动 ss 客户端
nohup sslocal -s your_server_ip -p your_server_port -l 1080 -k your_server_passwd -t 600 -m rc4-md5 > /dev/null 2>&1 &

[linux] docker容器save、load

2019-09-22

1、查看要保存的镜像

2、保存镜像

docker save zeven0707/jenkins:v4 -o /data/jenkins.tar

docker load -i jenkins.tar

docker run -dti --names=jenkins -p 10000:8080 -p 50000:50000 zeven0707/jenkins:v4
